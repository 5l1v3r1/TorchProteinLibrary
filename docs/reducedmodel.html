ReducedModel

This module contains implementation of angles to coordinates conversion for reduced protein models.

class ReducedModel.Angles2Backbone

This layer converts angles to coordinates of backbone atoms. The position of the $i$-th atom in the chain is obtained by: $$\mathbf{x}_i = B_0 B_1 \cdots B_i \mathbf{x}_0$$ where $\mathbf{x}_0$ is the reference frame origin and $B_i$ is the transformation matrix, parametrized by dihedral angles: $$B(\alpha, \beta, d) = \begin{bmatrix} \cos(\beta) & \sin(\alpha)\sin(\beta) & \cos(\alpha)\sin(\beta) & d\cos(\beta) \\ 0 & \cos(\alpha) & -\sin(\alpha) & 0 \\ -\sin(\beta) & \sin(\alpha)\cos(\beta) & \cos(\alpha)\cos(\beta) & -d\sin(\beta) \\ 0 & 0 & 0 & 1 \end{bmatrix}$$ The parameters of transformation depend on the index as follows:
  • $N$-$C_{\alpha}$ bond: $i = 3k + 1$, then $ B_i(\alpha, \beta, d) = B_i(\phi_{k}, \pi - 1.9391, 1.460)$
  • $C_{\alpha}$-$C$ bond: $i = 3k + 2$, then $ B_i(\alpha, \beta, d) = B_i(\psi_{k}, \pi - 2.061, 1.525)$
  • Peptide bond: $i = 3k$, then $ B_i(\alpha, \beta, d) = B_i(\pi, \pi - 2.1186, 1.330)$
During the forward pass, we save the products of transformation matrices for each atom: $$A_i = B_0 B_1 \cdots B_i$$ Notice that atom $3j$ is always N, atom $3j + 1$ is always CA, and atom $3j+2$ is always C. Thus transformation matrices $B_{3j+1}$ and $B_{3j+2}$ depend on the angles $\phi_j$ and $\psi_j$ correspondingly. During the backward pass, we first compute gradients of each atomic coordinate with respect to each angle: $$ \frac{\partial \mathbf{x}_i}{\partial \phi_j} = B_0 \cdots \frac{\partial B_{3j+1}}{\partial \phi_j} \cdots B_i \mathbf{x}_0 $$ $$ \frac{\partial \mathbf{x}_i}{\partial \psi_j} = B_0 \cdots \frac{\partial B_{3j+2}}{\partial \psi_j} \cdots B_i \mathbf{x}_0 $$ We can rewrite these expressions using the matrices $A$, that we saved during the forward pass: $$ \frac{\partial \mathbf{x}_i}{\partial \phi_j} = A_{3j} \frac{\partial B_{3j+1}}{\partial \phi_j} A_{3j+1}^{-1} A_{i} \mathbf{x}_0 $$ $$ \frac{\partial \mathbf{x}_i}{\partial \psi_j} = A_{3j+1} \frac{\partial B_{3j+2}}{\partial \psi_j} A_{3j+2}^{-1} A_{i} \mathbf{x}_0 $$ We also notice that transformation matrices have two properties:
  • Product of transformation matrices is a transformation matrix, therefore matrices $A$ are transformation matrices
  • Inverse of a transformation matrix can be computed as follows:
    If $ T = \begin{bmatrix} R & \mathbf{x} \\ 0 & 1 \end{bmatrix} $, then $T^{-1} = \begin{bmatrix} R^{T} & -R^{T}\mathbf{x} \\ 0 & 1 \end{bmatrix} $
Therefore we can compute all the derivatives simultaneously on GPU. To compute the derivatives of input of the layer with respect to the output we just have to calculate the following sums: $$\sum_i \frac{\partial L_k}{\partial \mathbf{x}_i}\cdot \frac{\partial \mathbf{x}_i}{\partial \phi_j}$$ and $$\sum_i \frac{\partial L_k}{\partial \mathbf{x}_i}\cdot \frac{\partial \mathbf{x}_i}{\partial \psi_j}$$

Example


import torch
from TorchProteinLibrary import ReducedModel
import numpy as np
import matplotlib.pylab as plt
import mpl_toolkits.mplot3d.axes3d as p3


if __name__=='__main__':
	a2b = ReducedModel.Angles2Backbone()
		
	#Setting conformation to alpha-helix
	N = 20
	num_aa = torch.zeros(1, dtype=torch.int, device='cuda').fill_(N)
	angles = torch.zeros(1, 2, N, dtype=torch.float, device='cuda')
	angles.data[:,0,:] = -1.047
	angles.data[:,1,:] = -0.698

	#Converting angles to coordinates
	coords = a2b(angles, num_aa)
	num_atoms = num_aa*3
	
	#Resizing coordinates array for convenience (to match selection mask)
	N = int(num_atoms.data[0])
	coords.resize_(1, N, 3)
	
	#Plotting backbone with the blue line
	sx, sy, sz = coords[0,:,0].cpu().numpy(), coords[0,:,1].cpu().numpy(), coords[0,:,2].cpu().numpy()

	fig = plt.figure()
	plt.title("Reduced model")
	ax = p3.Axes3D(fig)
	ax.plot(sx,sy,sz, 'r-', label = 'backbone')
	ax.legend()
	ax.set_xlim(-30, 0)
	ax.set_ylim(-30, 0)
	ax.set_zlim(-30, 0)
	# plt.show()
	plt.savefig("ExampleAngles2Backbone.png")
