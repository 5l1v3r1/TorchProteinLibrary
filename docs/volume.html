Volume

This module contains layers working with volumetric representation of proteins.

class Volume.TypedCoords2Volume(box_size=120)

Computes 3D atomic densities of each atom type.

class Volume.SelectVolume(box_size_bins=120, box_size_ang=120)

Computes feature columns from set of volumes using coordinates, scaled according to proportion of initial volume size (box_size_bins) and current volume size (box_size_ang).

class Volume.VolumeConvolution(num_features)

Computes parametric weighted correlation between two sets of volumes of the same size.
